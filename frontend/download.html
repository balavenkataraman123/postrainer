Download Postrainer

Download Instructions

Use a virtual environment like Anaconda. The requirements are:

Python == 3.7.9
Tensorflow == 2.8.0
Mediapipe == 0.8.9.1
Opencv-python == 4.5.5.64
Protobuf == 3.20.1

To use, download and extract distribcopy.rar, download the day's workout.txt file from the website to the same directory, and run modeltest1.py. When running, the program generates a score.txt file the first time, (and later appends to it) where scores for workouts will be logged over time for you to keep track of. If any error occurs, contact BariumLanthanum#7853 or Trent#4680 on discord.
